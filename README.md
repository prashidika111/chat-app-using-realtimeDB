# Chat Rooms Application

A real-time chat application built using Firebase Realtime Database and Cloudinary that allows users to create chat rooms, exchange messages, share images, and communicate instantly through both group and private conversations.
This project was developed as an improved version of a previous chat application, with a focus on better state management, real-time data synchronization, scalable message handling, and an improved user experience. Users can create chat rooms and receive auto-generated room codes that others can use to join and participate in conversations. Usernames are generated automatically, allowing users to quickly start interacting without account creation.
The application leverages Firebase Realtime Database for real-time messaging, room management, and online presence tracking, while Cloudinary is used for image storage and delivery. The project provided hands-on experience with real-time systems, third-party service integration, state management, and designing responsive user interfaces.

## Features

* Real-time group messaging
* Real-time private messaging
* Chat room creation and joining through room codes
* Auto-generated usernames
* Image sharing with Cloudinary integration
* Online user presence tracking
* Unread message indicators
* Responsive and user-friendly interface
